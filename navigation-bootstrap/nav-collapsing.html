Collapsing Navigation

Look at the page source to see how the nav is organized.

As you resize the browser and the viewport decreases, the nav disappears and a menu icon appears at the upper right. Nav come back when you click the icon.

There are some additional classes. The ul is in a div with the classes collapse and navbar-collapse. It has an id of collapsemenu as well. Additional, there is a div right above that that has a class of navbar-header. It has a button that has a few classes and attributes: classes of navbar-toggle and collapsed, attributes of data-toggle set to collapse and data-target set to #collapse menu, the id. There are also 3 spans with a class of icon-bar.
